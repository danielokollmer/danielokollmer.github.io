Erhobene Daten

Diese Anwendung erhebt und speichert keine personenbezogenen Daten. Im Einzelnen gilt:

  • Keine Registrierung, kein Nutzerkonto, kein Login
  • Keine Cookies
  • Kein Tracking, kein Analytics, keine Werbung
  • Keine Übertragung von Daten an externe Server
  • Quiz-Ergebnisse und Lernfortschritt werden nur im Arbeitsspeicher des Browsers gespeichert und bei Schließen der App automatisch gelöscht
  • Keine dauerhafte lokale Speicherung (kein localStorage, kein sessionStorage)

Diese Daten werden von GitHub gemäß deren Datenschutzrichtlinie verarbeitet und liegen nicht in meiner Kontrolle. Weitere Informationen: GitHub Datenschutzerklärung
